# 用戶端核心流程

本子域承接 `CJ` 面向普通用戶的核心業務流程入口，覆蓋 Web 與 Mobile 主鏈路。

當前正式文檔：

1. [00-用戶端核心流程總覽.md](./00-用戶端核心流程總覽.md)

當前重點包括：

1. 快速體驗、正式案件、聊天室、Repair Journey
2. 心理訪談、心理畫像、個人資料與配對相關主流程
3. 與前台主流程直接相關的頁面入口與狀態轉移

本子域不回答：

1. API 是否在用、候選廢棄或已確認廢棄
2. 管理端治理、平台監控與運營配置
3. 共用 package 落點與工程分層
