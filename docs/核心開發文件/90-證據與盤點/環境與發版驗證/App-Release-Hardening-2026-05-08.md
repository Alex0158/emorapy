## Native Maestro Evidence 格式

正式 native Maestro execution 應使用：

```bash
npm --prefix mobile run maestro:native
```

或只跑指定 flow：

```bash
npm --prefix mobile run maestro:native -- --flow=00-public-navigation-smoke.yaml
```

`mobile/scripts/run-maestro-native.mjs` 會先執行 `maestro:check` 等效 static gate，再執行 `native:check:strict`，然後逐一執行 `maestro test maestro/*.yaml`。成功或失敗都會輸出 `App-Native-Maestro-*.json`，包含 `app_id`、`static_gate`、`native_readiness`、每個 flow 的 exit code / duration / stdout tail / stderr tail。2026-05-08 已在 iOS 26.4.1 / iPhone 17 simulator 取得 7/7 pass；後續仍需把 physical device / TestFlight / provider side-effect evidence 另外留證。
