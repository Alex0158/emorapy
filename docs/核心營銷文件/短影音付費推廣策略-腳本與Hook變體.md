## 2. 主力一：可負擔替代路徑（2 支）

### 2.1 變體 E-α（Hook：價格 + 多期）


| 時段     | 台詞／字幕                                                 |
| ------ | ----------------------------------------------------- |
| 0–2s   | **心理諮商不是不好——是一節幾千，還要談很多週。**                           |
| 2–6s   | 學生、上班族，不是不想整理關係，是**每一次都負擔很重**。                        |
| 6–12s  | Emorapy 讓你用**極低成本**，先把這次爭執**用第三方角度講清楚**：雙方說法、AI 整理成听得懂的說法。 |
| 12–15s | **（合規字幕，固定）** 見合規文件「標準短版 A」。                          |
| 15–18s | 先試**快速體驗**，再決定要不要走下一步。                                |
| 18–20s | **Emorapy**——先把爭執講成別人也听得懂。                                 |


### 2.2 變體 E-β（Hook：只要「這一輪」講清楚）


| 時段     | 台詞／字幕                                  |
| ------ | -------------------------------------- |
| 0–2s   | **一節幾千、還要好幾週才進入狀況——如果你只是要把「這次吵的」講清楚？** |
| 2–6s   | 你不一定要立刻走完整諮商流程，可以先要**結構化調解**。          |
| 6–12s  | Emorapy：AI 當第三方，**整理責任敘事**，再給和好方向。          |
| 12–15s | **（合規字幕）** 標準短版 A。                     |
| 15–20s | 快速體驗在連結／簡介；**Emorapy**。                     |

## 3. 主力二：第三方把兩邊話聽完（2 支）

### 3.1 變體 T-α（Hook：男友不懂我）


| 時段     | 台詞／字幕                                             |
| ------ | ------------------------------------------------- |
| 0–2s   | **為什麼他永遠聽不懂我在氣什麼？**                               |
| 2–6s   | 你講感受，他講道理；你覺得被忽視，他覺得你無理取鬧。                        |
| 6–14s  | **Emorapy** 讓雙方先說清楚，AI 當第三方整理成**听得懂的說法**，還有**和好的下一步**。 |
| 14–18s | 不用先說服他——你先試**快速體驗**。                              |
| 18–20s | **Emorapy**，給爭執一個下得來的台階。                               |


### 3.2 變體 T-β（Hook：互動迴圈）


| 時段     | 台詞／字幕                                     |
| ------ | ----------------------------------------- |
| 0–2s   | **你們吵到最後，像在吵誰比較爛——其實可能是同一個互動迴圈。**         |
| 2–6s   | 重點不是誰贏，是**兩邊說法有沒有被完整聽見**。                 |
| 6–14s  | **Emorapy** 不站隊罵人，先把雙方陳述攤開，AI 幫你看**責任比例**與怎麼收。 |
| 14–18s | 免註冊，**快速體驗**一輪。                           |
| 18–20s | **Emorapy**——不是誰輸誰贏，是讓關係听得懂。                   |

## 4. 主力三：低門檻快速體驗（2 支）

### 4.1 變體 L-α（Hook：貴 vs 吵）


| 時段     | 台詞／字幕                                         |
| ------ | --------------------------------------------- |
| 0–2s   | **情侶諮商很貴，吵架很便宜——但後者超傷關係。**                    |
| 2–6s   | 你不用先報名、先拉對方，**先試一輪**看自己值不值得往下走。               |
| 6–14s  | **Emorapy 快速體驗**：**免註冊**，AI 用「法庭式結構」幫你把爭執**整理一次**。 |
| 14–18s | 連結在簡介／下方，**30 秒**決定要不要繼續。                     |
| 18–20s | **Emorapy**。                                       |


### 4.2 變體 L-β（Hook：叫不動伴侶）


| 時段     | 台詞／字幕                          |
| ------ | ------------------------------ |
| 0–2s   | **伴侶諮商要兩個人都願意——你叫不動他，怎麼辦？**    |
| 2–6s   | 你可以**一個人先開始**：把故事講完整，再談要不要一起用。 |
| 6–14s  | **Emorapy 快速體驗**：零註冊，跑完一輪**結構化流程**。 |
| 14–18s | 點連結試試。                         |
| 18–20s | **Emorapy**——先把你的這邊說清楚。             |
