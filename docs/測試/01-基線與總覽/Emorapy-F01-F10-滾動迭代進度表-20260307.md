# Emorapy F01-F10 滾動迭代進度表

日期：2026-03-07  
用途：把 `F01 -> F10` 的測試開發節奏、文檔產出、回歸結果與缺陷閉環放在同一張表中持續更新。

## 使用規則

1. 每完成一個功能，必須同步更新本表。
2. `文檔狀態` 需對齊 `01/02/03/04/05` 是否已產出。
3. `測試狀態` 需對齊當輪實際測試結果，而不是理想狀態。
4. 若當輪暴露真缺陷，`缺陷/風險` 不得留空。
5. 若表內仍沿用歷史名稱，請以功能 ID 對齊核心 SSOT：
   - `F01` = 快速體驗、結果與升格
   - `F04` = 判決生成、查看與接受
   - `F06` = 心理訪談、結果與心理畫像
   - `F07` = Chat 房間、邀請與轉判決
   - `F09` = 身份認證與個人資料管理
   - `F10` = Admin 運維與治理
