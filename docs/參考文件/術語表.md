# JobPortal 術語表

本文件定義核心術語，避免跨文檔歧義。

## 角色

| 術語 | 定義 |
|------|------|
| 學生（student） | 求職端用戶，使用職位搜索、申請、收藏、簡歷等能力。 |
| 僱主（employer） | 企業端用戶，使用職位發布、申請管理、企業中心等能力。 |
| 管理員（admin） | 平台管理角色，負責審核、系統配置、統計分析等能力。 |

## 接口級別

| 術語 | 定義 | 例子 |
|------|------|------|
| 平台級 API | 由通用平台能力提供，JobPortal 直接調用或適配使用。 | `/api/openApi/*`、`/api/permission/*` |
| JobPortal 級 API | 由 JobPortal 業務域直接提供與維護。 | `/api/jp/*` |

## 狀態與流程

| 術語 | 定義 |
|------|------|
| 審核（audit） | 對註冊、企業、職位等資料執行通過/拒絕的業務流程。 |
| 流程節點 | 流程中的關鍵業務步驟，可映射到 API、頁面或定時任務。 |
| 主規格 | 作為單一真相來源（SSOT）的文檔入口。 |
| 映射（Mapping） | API、功能、頁面之間的可追溯關係表。 |

## 文檔治理

| 術語 | 定義 |
|------|------|
| 單一真相來源（SSOT） | 在同一主題下唯一可被依賴的正式文檔。 |
| 歸檔（Archived） | 不再作為主規格使用，但保留歷史追溯價值的文檔。 |
| 溯源鏈接 | 從總覽文檔鏈接到詳細規格文檔的引用關係。 |
