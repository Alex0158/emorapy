## 2. 驗證口徑

本輪只接受以下證據作為「已收斂」依據：

- 前端 E2E 通過，且能覆蓋真實入口與跨頁 handoff。
- 前端頁面/路由/狀態恢復測試通過，且覆蓋 retry、fallback、權限、回跳。
- 後端 integration 或 route/service 測試通過，且覆蓋權限矩陣、主鏈路或關鍵錯誤碼。
- 若未跑成，需明確區分為「功能缺陷」或「環境/證據缺口」。
