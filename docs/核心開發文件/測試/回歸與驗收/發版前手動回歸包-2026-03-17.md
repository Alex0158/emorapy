## 3. P01 快速體驗閉環

目標：

- 驗證 `quick create -> result -> login/register -> claim-session -> 回訪原結果`。

操作：

1. 以匿名身份進入 `/quick-experience/create` 建立 quick case。
2. 進入 result 頁，確認判決/等待態可見。
3. 從 result 頁走登入或註冊。
4. 完成 `claim-session`。
5. 確認回到原 `result` 頁，並可正常查看內容。
