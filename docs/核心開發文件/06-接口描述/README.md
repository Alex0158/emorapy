本子域不回答：

1. 某個接口是否在用、候選廢棄或已確認廢棄
2. 某個功能或頁面的正式存在性
3. 跨端主流程的高位正文

使用順序：

1. 先查 [../全接口清單-主文檔.md](../全接口清單-主文檔.md) 確認接口狀態。
2. 再進入本目錄查看模組內契約、業務規則、限流與回歸風險。
3. 若要做影響分析，回查 [../接口-功能-頁面-Mapping.md](../接口-功能-頁面-Mapping.md)。
