# 文件收斂

本目錄只承接 `核心開發文件/` 的文檔治理資料，不承接業務規格正文。

它回答的是：

1. 這個倉庫的文檔應怎樣收斂
2. 哪些文件屬於活躍治理層
3. 某一批文檔為何被保留、下沉、遷移或退出

它不回答：

1. 現行業務流程是什麼
2. 現行接口、頁面、功能與角色規則是什麼
3. 測試驗收標準正文是什麼

活躍文件固定如下：

1. `README.md`
2. `00-CJ-文檔治理與同步規則.md`
3. `01-CJ-文檔收斂台賬與批次索引.md`
4. `02-CJ-全項目Markdown收斂台賬-2026-04-18.md`

其餘文件默認按歷史治理記錄理解。
