## 命名

`YYYY-MM-DD-{腳本軸}_{簡短slug}.md`

## 產出方式

使用 Cursor Skill **`cj-short-video-campaign-brief`**（專案路徑：`.cursor/skills/cj-short-video-campaign-brief/`），請 Agent 依 [campaign-doc-template.md](../../../../.cursor/skills/cj-short-video-campaign-brief/campaign-doc-template.md) 結構產出並寫入本目錄。進階規則（合規決策樹、Prompt 五層、平台微差）見同目錄 [reference.md](../../../../.cursor/skills/cj-short-video-campaign-brief/reference.md)。
