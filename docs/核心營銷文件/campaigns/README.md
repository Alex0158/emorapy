# 付費短片單案文檔（一方案一檔）

本目錄存放 **單支／單方案** 的完整創意與生成用 Prompt 文檔（Markdown）。

## 命名

`YYYY-MM-DD-{腳本軸}_{簡短slug}.md`

範例：`2026-03-21-E-alpha_counseling-cost.md`

## 產出方式

依本目錄既有單案結構產出，一案一檔，將單支素材的策略、合規、分鏡與 Prompt 全部收斂在對應 Markdown 內。

若需回看舊 Prompt 模板、歷史 Skill 版本或早期參考資料，請至根級 `歸檔/` 查歷史資料；這些內容不再作當前營銷依據。
