- `docs/核心開發文件/20-App端/03-App完整版本開發Roadmap.md` 現在同時承接 Roadmap 裁決與大量單次命令 / artifact 細節。

## 文件偏差

目前偏差不是「App 狀態判斷錯誤」，而是「正式規格承載粒度過細」。這會讓讀者難以分辨：

1. 長期有效的產品 / 工程裁決。
2. 一次性驗收記錄。
3. release blocker 的當前狀態。
4. 已完成後仍保留的歷史修復過程。
