## 4. 當前文檔維護規則

1. **路由/API 改動**：先改 `backend/src/routes/*` 或 `frontend/src/router/index.tsx`，再同步 docs  
2. **核心清單優先更新**：
   - `docs/功能特性清單.md`
   - `docs/前端設計/02-路由與頁面結構設計.md`
   - `docs/前端設計/08-接口一覽表.md`
   - `docs/後端設計/03-API設計.md`
3. 本文件僅描述「存在且使用中」結構，不保留已刪除或歷史目錄占位
