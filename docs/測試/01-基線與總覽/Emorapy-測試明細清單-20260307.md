## 統計口徑

- 後端範圍：`backend/tests/**/*.test.ts`。
- 前端單元/元件範圍：`frontend/src/**/*.{test,spec}.{ts,tsx}`。
- 前端 E2E 範圍：`frontend/e2e/**/*.e2e.ts`。
- Admin E2E 範圍：`e2e/**/*.e2e.ts`。
- 案例數統計：統一以 `it()` / `test()` 出現次數作近似盤點；本數據可作基線，不視為最終審計口徑。

- 名稱映射：若表內仍出現歷史名稱，請以功能 ID 對齊核心 SSOT；

  - `F10` = Admin 運維與治理
