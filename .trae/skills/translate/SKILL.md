# translate v1.0

## 目的
跨前後端提取文本並補齊翻譯資源，輸出統計與風險提示。

## 何時觸發
- 國際化改造或日常自查

## 步驟
1. 掃描前後端源碼，提取待國際化文本，使用 AI 建議翻譯；提供源碼上下文。
2. 載入資源文件，比對缺失，生成 key/value；支持導出腳本/映射表。
3. 批量寫入（vue-i18n/react-intl 等）；檢測多義詞/風險，輸出需人工列表。
4. 匯總統計（新增數 + 腳本），考慮翻譯效率。

## 輸出
- 匯總統計
- confidence: high/medium/low
