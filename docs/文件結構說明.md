## 🎯 使用建議

### 開始開發前
1. 閱讀 `README.md` 和 `QUICK_START.md`
2. 閱讀 `10-MVP開發計劃.md`
3. 閱讀 `11-開發環境配置.md`
4. 閱讀 `ENVIRONMENT.md`

### 開發過程中
1. 參考 `前端設計/` 和 `後端設計/` 文檔
2. 參考 `後端設計/12-接口建設規範.md`
3. 參考 `INTEGRATION.md` 前後端集成

### 部署時
1. 閱讀 `發佈流程指引.md`
2. 閱讀 `backend/DEPLOYMENT.md`
3. 執行 `scripts/validate-env.sh` 驗證環境
