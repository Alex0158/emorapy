## 3. 文件類型與對應

| 你要找的內容 | 優先看 |
|---|---|
| 功能/特性清單 | `docs/核心開發文件/功能特性清單.md` |
| 頁面清單 | `docs/核心開發文件/頁面清單.md` |
| 業務流程（代碼落地） | `docs/核心開發文件/業務流程整合.md` |
| API 清單 | `docs/核心開發文件/全接口清單-主文檔.md` + `docs/核心開發文件/接口描述/` |
| 新人啟動/聯調 | `docs/QUICK_START.md` + `docs/INTEGRATION.md` |
| 環境與部署 | `docs/ENVIRONMENT.md` + `docs/發佈流程指引.md` |

---

## 4. 維護規則（避免再次過時）

1. 改路由先改代碼，再同步：
   - 前端 `router/index.tsx` -> `docs/核心開發文件/頁面清單.md`
   - 後端 `routes/*.ts` -> `docs/核心開發文件/全接口清單-主文檔.md`
2. 涉及跨模塊流程時，至少同步：
   - `功能特性清單`
   - `INTEGRATION`
3. 快查文檔（`backend/API.md`）不放字段細節，只做索引；字段級契約統一維護在 `核心開發文件` 主文檔。
