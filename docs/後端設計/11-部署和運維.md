## 📦 環境配置

### 環境變量

**文件**：`.env.example`

```env
# 服務器配置
NODE_ENV=production
PORT=3001

# 數據庫配置
DATABASE_URL=postgresql://user:password@host:5432/dbname

# JWT配置
JWT_SECRET=your-secret-key-change-in-production
JWT_EXPIRES_IN=7d

# OpenAI配置
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-3.5-turbo
OPENAI_MAX_TOKENS=2000
OPENAI_DAILY_LIMIT=1000

# 郵件配置
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-password
EMAIL_FROM=noreply@motherbearcourt.com

# 文件存儲配置（實現以 UPLOAD_DIR、FILE_BASE_URL 為準，見 backend/.env.example）
UPLOAD_DIR=./uploads
FILE_BASE_URL=https://your-backend-domain.com

# 前端URL
FRONTEND_URL=https://motherbearcourt.com

# 允許的來源
ALLOWED_ORIGINS=https://motherbearcourt.com,https://www.motherbearcourt.com

# 日誌級別
LOG_LEVEL=info
```

## 🏗️ 部署流程

### 1. 準備工作

```bash
# 1. 確保代碼已提交到Git
git add .
git commit -m "Ready for deployment"
git push origin main

# 2. 運行測試
npm run test

# 3. 構建項目（後端使用 build:prod）
cd backend && npm run build:prod
```

### 2. Railway部署

**步驟**：

1. 在Railway創建新項目
2. 連接GitHub倉庫
3. 配置環境變量
4. 部署

**Railway配置**：

```json
{
  "build": {
    "builder": "NIXPACKS"
  },
  "deploy": {
    "startCommand": "npm start",
    "restartPolicyType": "ON_FAILURE",
    "restartPolicyMaxRetries": 10
  }
}
```

### 3. Render部署

**步驟**：

1. 在Render創建新Web服務
2. 連接GitHub倉庫
3. 配置構建命令：`npm install && npm run build`
4. 配置啟動命令：`npm start`
5. 配置環境變量
6. 部署

## 🗄️ 數據庫部署

### Supabase設置

1. 創建Supabase項目
2. 獲取數據庫連接字符串
3. 運行遷移：

```bash
# 設置數據庫URL
export DATABASE_URL="postgresql://..."

# 運行遷移
npx prisma migrate deploy
```

### 數據庫遷移

```bash
# 開發環境：創建遷移
npx prisma migrate dev --name migration_name

# 生產環境：應用遷移
npx prisma migrate deploy
```

## 🚨 故障處理

### 常見問題

**1. 服務無法啟動**

```bash
# 檢查日誌
railway logs

# 檢查環境變量
railway variables

# 檢查數據庫連接
npx prisma db pull
```

**2. 數據庫連接失敗**

- 檢查DATABASE_URL是否正確
- 檢查數據庫是否運行
- 檢查網絡連接

**3. AI服務錯誤**

- 檢查OPENAI_API_KEY是否有效
- 檢查API限額
- 檢查網絡連接

**實現對齊**：後端構建為 `npm run build:prod`；環境變量以 `backend/.env.example` 為準（含 UPLOAD_DIR、FILE_BASE_URL）；完整流程見 [發佈流程指引](../發佈流程指引.md)、[ENVIRONMENT](../ENVIRONMENT.md)。
