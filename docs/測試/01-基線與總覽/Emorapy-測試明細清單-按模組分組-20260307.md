# Emorapy 測試明細清單（按模組分組）

日期：2026-03-07  
用途：把現有測試資產對齊 Emorapy 功能主表（F01-F10），形成後續專項測試設計與排期的模組台賬。

## 統計口徑

- 與 `Emorapy-測試明細清單-20260307.md` 相同。
- 本版以功能主表 `F01-F10` 為主分組，`F11-F14` 與共用基礎能力暫併入「候選功能 / 通用基礎」。
- 單一測試檔若同時覆蓋多個功能，本版先按主要責任歸屬。後續若有需要，再在專項文檔中做多歸屬標註。
- `案例數` 與 `檔案數` 以本文件各模組明細表逐行重算為準。
- `覆蓋率` 為模組級測試覆蓋估算，不等於程式碼行覆蓋率；分母只計入核心文件中已定義且程式碼中已落地的責任點。
- `覆蓋率` 以六個維度盤點：`頁面/場景`、`API`、`流程步驟`、`狀態轉移`、`資料庫交互`、`高風險分支`；按 `covered=1`、`partial=0.5`、`uncovered=0` 換算百分比。
- `候選功能 F11-F14 / 通用基礎` 的覆蓋率只把已落地且被主流程實際引用的責任點納入分母，不把純候選或保留能力硬算進去。
