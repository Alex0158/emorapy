## 高風險回歸清單（接口視角）


| 風險類型             | 涉及接口                                                                        | 回歸關注點                                                 |
| ---------------- | --------------------------------------------------------------------------- | ----------------------------------------------------- |
| Session 與 JWT 混用 | `/sessions/`*、`/cases/*`、`/chat/*`、`/auth/claim-session`                    | `X-Session-Id` 與 `Authorization` 並存時授權優先級、過期刷新與 quick 升格行為 |
| 匿名房主 session 匹配 | `/api/v1/chat/rooms`、`/api/v1/chat/rooms/:roomId/invites`                      | 匿名 A 房主建立 invite 時，`room.session_id` 必須與當前 `canonical session_id` 對齊 |
| SSE 長連線          | `/chat/rooms/:roomId/stream`、`/streams/:scopeType/:scopeId`、`/interview/:id/respond`、`/interview/:id/skip` | 中斷重連、超時、事件解析錯誤是否導致 UI 卡死                              |
| 聊天轉判決冪等          | `/chat/rooms/:roomId/request-judgment`、`/judgments/generate/:id`            | 重複點擊、房間狀態競態、舊 link 復用邏輯；前端 `request-judgment` 採 `180000ms` 請求窗口，超時後須回查 `judgment-status` 而非直接判失敗；`included_message_ids` 只允許 `user_text + visibility_scope=all` 且遵守 join-time 可見窗 |
| Case 詳情授權分流 | `/api/v1/cases/:id`、`/api/v1/cases/:id/judgment` | `collaborative + session_id=null` 不得被誤判為 session-only；必須按當事人 JWT 校驗 |
| 文件上傳鏈路           | `/cases/:id/evidence`、`/uploads/*`                                          | MIME 驗證、大小限制、下載授權、Session 案件隔離；`FormData` 必須由瀏覽器自動注入 multipart boundary，且本地 `localhost/127.0.0.1/::1` 視同站 |
| 管理員權限            | `/admin/*`                                                                  | permission matrix 是否正確攔截（`ops:read`、`config:write` 等） |

| 錯誤碼 | HTTP | 類型 | 典型觸發接口 | 首選恢復動作 |
|---|---:|---|---|---|
| `UNAUTHORIZED` | 401 | 鑑權 | 任何需 token 的 `/api/v1/*` | 清 token 後導向登入 |
| `TOKEN_EXPIRED` | 401 | 鑑權 | `/api/v1/user/*`、`/api/v1/cases` | 走 refresh 或重新登入 |
| `FORBIDDEN` | 403 | 權限 | `/api/v1/admin/*`、`/api/v1/chat/*` | 顯示無權限並禁用操作 |
| `INVALID_CREDENTIALS` | 401 | 認證 | `POST /api/v1/auth/login`、`POST /api/v1/admin/login` | 留在登入頁提示重試 |
| `VALIDATION_ERROR` | 400 | 參數校驗 | 幾乎所有 POST/PUT/PATCH | 定位字段錯誤並保留輸入 |
| `INVALID_EMAIL` | 400 | 參數校驗 | `POST /api/v1/auth/register` | 前端 email 字段高亮 |
| `WEAK_PASSWORD` | 400 | 參數校驗 | 註冊/重設密碼 | 顯示密碼規則 |
| `INVALID_CODE` | 400 | 驗證碼 | `POST /api/v1/auth/verify-email` | 允許重新輸入 |
| `CODE_EXPIRED` | 400 | 驗證碼 | 驗證碼流程 | 引導重新發碼 |
| `SESSION_ID_REQUIRED` | 400 | Session | `/api/v1/cases/by-session`、`/api/v1/chat/rooms` | 補建 quick session |
| `INVALID_SESSION_ID` | 400 | Session | `/api/v1/auth/claim-session` | 清理無效 session 後重試 |
| `SESSION_EXPIRED` | 401 | Session | `/api/v1/cases/*`、`/api/v1/chat/*` | refresh session 或回入口 |
| `NOT_FOUND` | 404 | 資源 | `/api/v1/cases/:id`、`/api/v1/judgments/:id` | 回上游頁重選資源 |
| `EMAIL_EXISTS` | 409 | 衝突 | `POST /api/v1/auth/register` | 提示改用登入/找回密碼 |
| `ALREADY_PAIRED` | 409 | 業務衝突 | `POST /api/v1/pairing/join` | 導向配對狀態頁 |
| `CONFLICT` | 409 | 衝突 | Prisma 唯一鍵衝突泛化 | 引導刷新資料再提交 |
| `CASE_NOT_READY` | 422 | 業務前置條件 | `POST /api/v1/chat/rooms/:roomId/request-judgment` | 提示補充內容後重試 |
| `CASE_NOT_EDITABLE` | 422 | 狀態限制 | `PUT /api/v1/cases/:id` | 顯示唯讀態 |
| `JUDGMENT_EXISTS` | 409 | 業務衝突 | `POST /api/v1/judgments/generate/:id` | 直接拉既有 judgment |
| `JUDGMENT_FAILED` | 409 | 業務失敗 | 判決流程 | 顯示可重試操作 |
| `FILE_TOO_LARGE` | 413 | 上傳 | `POST /api/v1/cases/:id/evidence` | 壓縮或更換文件 |
| `INVALID_FILE_TYPE` | 400 | 上傳 | 證據/頭像上傳 | 改為支持格式 |
| `TOO_MANY_FILES` | 400 | 上傳 | 多文件上傳 | 分批上傳 |
| `INVALID_FILE_FIELD` | 400 | 上傳 | Multer 欄位名不匹配 | 修正 multipart 欄位 |
| `UPLOAD_ERROR` | 400 | 上傳 | Multer 兜底錯誤 | 重試並檢查網路/文件 |
| `RATE_LIMIT_EXCEEDED` | 429 | 限流 | auth/chat/upload 等 limiter | 冷卻後重試 |
| `CONSENT_REQUIRED` | 403 | 訪談前置 | `POST /api/v1/interview/start` | 先完成 consent |
| `CONCURRENT_REQUEST` | 409 | 併發保護 | 訪談/chat 連續請求 | 等待上一請求完成 |
| `AI_CALL_FAILED` | 503 | 外部 AI（舊碼） | interview 舊鏈路保留 | 顯示可重試並回退 UI |
| `MAX_TURNS_REACHED` | 422 | 訪談規則 | `POST /api/v1/interview/:id/respond` | 引導 end 流程 |
| `TURN_TOO_FAST` | 429 | 訪談節流 | 訪談 respond/skip | 顯示節流倒數 |
| `SESSION_COMPLETED` | 409 | 訪談狀態 | `POST /api/v1/interview/:id/respond` | 跳轉結果頁 |
| `AI_SERVICE_ERROR` | 503 | 外部依賴 | AI 下游異常 | 降級提示/重試 |
| `DATABASE_ERROR` | 500 | 基礎設施 | DB 故障路徑 | 兜底錯誤提示與告警 |
| `EXTERNAL_SERVICE_ERROR` | 503 | 外部依賴 | 外部服務鏈路 | 退避重試 |
| `INTERNAL_ERROR` | 500 | 兜底 | 未分類錯誤 | 記錄 request_id 後回報 |
